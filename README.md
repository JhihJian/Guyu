## 功能期望
首先，最重要的是，个人助手应该能清楚我每天做了什么
包括但不限于，我每天吃了什么，买了什么，用了什么，去了哪儿，做了什么运动，聊了什么等等。一个全渠道的智能的、信息整合，并且不需要考虑隐私问题。
第二，是基于已经收集到的信息，能作出相关总结，或者报表一类的东西，例如饮食状态，运动情况等等
最后，基于内容和扩展专业知识，提供建议
